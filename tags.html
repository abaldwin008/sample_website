Important HTML tags

This page is designed as a quick reference for some of the common tages covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tablesem> via tablecode> and related tags.'

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
tr table row include a row of data
th table header make a new table header
td table data include a table data cell
ol ordered list make an ordered list
ul unordered list make an unordered list
li list item add a list item

Inline Elements
